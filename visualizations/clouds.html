Cloud Coverage Analysis

Explore the visual analysis of global weather cloud coverage across the Northern and Southern Hemispheres

This chart plots the percentage of cloud coverage at varying equitorial latitudes.

We are able to see that cloud coverage increases at about 20 degrees latitude in both the Northern and Sourthern hemispheres. However, beyond the Northern proximity to the equator there apears to be an even distribution of cloud coverage.

This chart plots the linear regression correlation of latitude to cloud coverage percentage within the Northern Hemisphere.

We are able to see that in the Northern Hemisphere there is no correlation between latitude and cloud coverage percentage. Meaning that cities located at various proximities to the equator have even distribution percentages of cloud coverage across these latitudes.

This chart plots the linear regression correlation of latitude to cloud coverage percentage within the Southern Hemisphere.

We are able to see that in the Southern Hemisphere there is a positive correlation between latitude and cloud coverage percentage. Meaning that cities which are located closer to the equator experience an increase in the percentatge of cloud coverage at these latitudes.
